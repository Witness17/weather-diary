# 🌦 Weather Diary - Дневник погоды

**Автор:** Пронюк Никита  
**GitHub:** https://github.com/Witness17/weather-diary.git

## 📖 Описание

**Weather Diary** — это графическое приложение на Python с использованием библиотеки Tkinter для ведения дневника погоды. Программа позволяет добавлять записи о погоде за разные дни, фильтровать их по дате и температуре, а также сохранять все данные в JSON-файл. Приложение помогает отслеживать погодные условия и анализировать изменения температуры.

## 🚀 Функциональность

- ✅ Добавление записей с указанием:
  - Даты
  - Температуры
  - Описания погоды
  - Наличия осадков (да/нет)
- ✅ Фильтрация записей:
  - По дате (выбор конкретного дня)
  - По температуре (только записи выше +10°C)
- ✅ Просмотр всех записей в табличном виде
- ✅ Сохранение записей в файл `weather_data.json`
- ✅ Автоматическая загрузка записей при запуске программы
- ✅ Проверка корректности ввода:
  - Дата в формате `ГГГГ-ММ-ДД`
  - Температура — числовое значение
  - Описание не пустое
- ✅ Использование системы контроля версий Git

## 🛠 Установка и запуск

### Требования
- Python 3.6 или выше
- Стандартные библиотеки: `tkinter`, `json`, `os`, `datetime`

### Запуск

```bash
# Клонировать репозиторий (замените ссылку на вашу)
git clone https://github.com/Witness17/weather-diary.git

# Перейти в директорию проекта
cd weather-diary

# Запустить программу
python main.py
